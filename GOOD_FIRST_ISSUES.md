# Good First Issues

These tasks are suitable for first-time contributors:

1. Add a new SOP document under `docs/knowledge/`.
2. Add a new Dashboard chart using ECharts.
3. Add a new Agent Tool with the unified tool result format.
4. Add a new Demo Prompt to the README.
5. Add a new export format for reports.
6. Add a new example scenario under `examples/`.
7. Improve the frontend copy for HITL approval cards.
8. Add unit tests for one dashboard endpoint.
